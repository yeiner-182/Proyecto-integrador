Explicación del ciclo while en Java

¿Qué es el ciclo while?

El ciclo while es una estructura de control que repite un bloque de instrucciones mientras una condición sea verdadera.

Estructura

while (condición) {
    // Código a ejecutar
}

Ejemplo básico

int contador = 1;

while (contador <= 5) {
    System.out.println("Contador: " + contador);
    contador++;
}

Este código imprime los números del 1 al 5.

¡Cuidado con los ciclos infinitos!

// Este código generará un ciclo infinito
int contador = 1;

while (contador <= 5) {
    System.out.println("Contador: " + contador);
    // contador++ está ausente
}

Comparación while vs do-while

Característica while do-while
Evaluación de la condición Antes de ejecutar Después de ejecutar
Ejecución mínima 0 veces 1 vez

Ejemplo práctico: menú

Scanner scanner = new Scanner(System.in);
int opcion = 0;

while (opcion != 4) {
    System.out.println("1. Ver productos\n2. Comprar\n3. Ayuda\n4. Salir");
    opcion = scanner.nextInt();

    switch (opcion) {
        case 1 -> System.out.println("Mostrando productos...");
        case 2 -> System.out.println("Proceso de compra...");
        case 3 -> System.out.println("Centro de ayuda...");
        case 4 -> System.out.println("Saliendo del sistema.");
        default -> System.out.println("Opción inválida.");
    }
}
scanner.close();
Resumen: El ciclo while en Java ejecuta un bloque de código mientras la condición sea verdadera. Es muy útil cuando no se sabe cuántas veces se repetirá el proceso.
